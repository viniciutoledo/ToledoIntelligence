Guia de Integração do Widget ToledoIA

Documentação completa para integrar o chat ToledoIA em qualquer plataforma

Visão Geral

O ToledoIA oferece múltiplas formas de integrar seu poderoso assistente de IA em sites e plataformas externas. Este guia fornece instruções detalhadas para cada método de integração, dependendo do seu caso de uso.

Widget via JavaScript

Método padrão para adicionar o chat em qualquer site ou aplicação web com controle total de personalização.

Ideal para: Sites pessoais, blogs, portfólios

Integração via iframe

Método simplificado para integrar o chat em plataformas que suportam incorporação de conteúdo externo.

Ideal para: LMS, intranet, plataformas com recursos iframe

Curseduca

Solução especializada para integração com a plataforma Curseduca para cursos online.

Ideal para: Professores e instituições que usam Curseduca

API Key Necessária: Todos os métodos de integração exigem uma API key válida do ToledoIA. Obtenha a sua no painel administrativo do ToledoIA na seção "Widgets".

Método 1: Widget JavaScript

Requisitos

  • Acesso à edição do código HTML do seu site
  • API key do ToledoIA (disponível no painel administrativo)

Implementação

Obter sua API key

Acesse o painel administrativo do ToledoIA, vá para a seção "Widgets" e crie um novo widget ou utilize um existente. Copie a API key gerada.

Adicionar o script ao seu site

Adicione o seguinte código ao final do seu arquivo HTML, logo antes da tag de fechamento </body>:

<script src="https://toledoia.com/widget.js" defer></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    ToledoIAWidget.init({
      apiKey: 'SUA_API_KEY',
      position: 'right',  // 'right' ou 'left'
      language: 'pt',     // 'pt' ou 'en'
      theme: 'light'      // 'light' ou 'dark'
    });
  });
</script>

Substitua SUA_API_KEY pela API key que você obteve no painel administrativo.

Opções de Configuração

Opção Tipo Padrão Descrição
apiKey String - Sua API key do ToledoIA (obrigatório)
position String 'right' Posição do widget na tela ('right' ou 'left')
language String 'pt' Idioma do widget ('pt' ou 'en')
theme String 'light' Tema do widget ('light' ou 'dark')
initialOpen Boolean false Se o widget deve iniciar aberto
hideHeader Boolean false Oculta o cabeçalho do widget

Método 2: Integração via iframe

A integração via iframe é ideal para plataformas onde você não tem acesso total ao código JavaScript ou precisa incorporar o ToledoIA em ambientes controlados.

Requisitos

  • Capacidade de adicionar um iframe em sua página/plataforma
  • API key do ToledoIA

Implementação Básica

<iframe 
  src="https://toledoia.com/embed/iframe?key=SUA_API_KEY" 
  width="100%" 
  height="600" 
  frameborder="0" 
  allow="microphone" 
  style="border: none; width: 100%; height: 600px;"
></iframe>

Substitua SUA_API_KEY pela API key que você obteve no painel administrativo.

Importante: Ao usar iframe, certifique-se de que sua API key está configurada para permitir o domínio onde o iframe está sendo incorporado, ou defina como "*" para permitir qualquer domínio (menos seguro).

Parâmetros da URL

Parâmetro Descrição
key Sua API key do ToledoIA (obrigatório)
hideHeader Define como 'true' para ocultar o cabeçalho do chat
language Idioma do widget ('pt' ou 'en')

Exemplo com Parâmetros

<iframe 
  src="https://toledoia.com/embed/iframe?key=SUA_API_KEY&hideHeader=true&language=en" 
  width="100%" 
  height="600" 
  frameborder="0" 
  allow="microphone" 
  style="border: none; width: 100%; height: 600px;"
></iframe>

Método 3: Integração com Curseduca

O ToledoIA oferece uma integração especial com a plataforma Curseduca, permitindo que professores incorporem a IA diretamente nos cursos.

Nota: Esta integração foi especialmente projetada para compatibilidade com a plataforma Curseduca, mas os mesmos princípios podem ser aplicados a outras plataformas LMS similares.

Passo a Passo para Integração

Obter sua API key

No painel administrativo do ToledoIA, acesse "Widgets" e crie ou selecione um widget existente. Copie a API key.

Adicionar ao Curseduca

No Curseduca, ao criar um novo item de conteúdo:

  1. Selecione a opção "Link Externo"
  2. No campo URL, adicione a seguinte URL formatada:
/embed?url=https%3A%2F%2Ftoledoia.com%2Fembed%2Fiframe%3Fkey%3DSUA_API_KEY

Substitua SUA_API_KEY pela sua API key do ToledoIA. Esta URL especial permite que o ToledoIA seja mostrado diretamente na página de curso, sem redirecionar para uma nova janela.

Configurações Adicionais

No Curseduca:

  • Marque a opção "Incorporar/Embed" se disponível
  • Defina a altura do iframe para pelo menos 600px
  • Recomenda-se permitir que o conteúdo ocupe a largura máxima disponível

Perguntas Frequentes

Como personalizar a aparência do widget?

Todas as personalizações visuais, como cores, avatar e mensagem de boas-vindas, devem ser feitas no painel administrativo do ToledoIA na seção "Widgets". Essas configurações são aplicadas automaticamente em qualquer integração que use a mesma API key.

O widget funciona em dispositivos móveis?

Sim, o ToledoIA é totalmente responsivo e se adapta a diferentes tamanhos de tela, incluindo smartphones e tablets.

Posso limitar o uso do widget a domínios específicos?

Sim, no painel administrativo do ToledoIA, você pode configurar domínios permitidos para cada widget. Isso impede que sua API key seja usada em sites não autorizados.

Como monitorar o uso do widget?

O painel administrativo do ToledoIA oferece estatísticas detalhadas de uso para cada widget, incluindo número de conversas, tópicos mais discutidos e muito mais.

O widget afeta o desempenho do meu site?

O ToledoIA foi projetado para ter impacto mínimo no desempenho do seu site. O script é carregado assincronamente e o widget só é inicializado quando necessário.

Solução de Problemas

O widget não aparece no meu site

  • Verifique se você está usando a API key correta
  • Confirme se o domínio do seu site está na lista de domínios permitidos
  • Verifique se há erros no console do navegador
  • Confirme se o widget está ativo no painel administrativo

As imagens não são exibidas corretamente

  • Verifique se o formato da imagem é suportado (JPG, PNG)
  • Confirme se a imagem não excede o tamanho máximo (50MB)
  • Tente redimensionar a imagem antes de fazer upload

O iframe não carrega no Curseduca ou outras plataformas

  • Confirme se você está usando a URL de incorporação correta
  • Verifique se a plataforma permite iframes de domínios externos
  • Teste se há restrições de CSP (Content Security Policy) na plataforma

Se persistirem os problemas após tentar as soluções acima, entre em contato com o suporte do ToledoIA para assistência adicional.
