Widget Embedado (Inline)

Para incorporar o widget ToledoIA diretamente em sua página, use o código abaixo:

<!-- Container para o widget -->
<div id="toledoia-container" style="position:relative; width:100%; height:600px;"></div>

<!-- Script do widget -->
<script src="https://toledoia.replit.app/widget.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    ToledoIAWidget.init({
      apiKey: "sua-chave-api-aqui",
      mode: "inline",
      targetElement: "#toledoia-container",
      hideHeader: true
    });
  });
</script>

Widget Flutuante

Para adicionar o widget ToledoIA como um botão flutuante em sua página, use o código abaixo:

<!-- Script do widget -->
<script src="https://toledoia.replit.app/widget.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    ToledoIAWidget.init({
      apiKey: "sua-chave-api-aqui",
      position: "bottom-right", // Outras opções: bottom-left, top-right, top-left
      initialOpen: false
    });
  });
</script>

Opções de Configuração

  • apiKey: Sua chave de API ToledoIA (obrigatória)
  • mode: "floating" (padrão) ou "inline"
  • targetElement: Seletor CSS ou elemento DOM para inserir o widget quando no modo inline
  • position: Posição do botão e widget no modo flutuante (bottom-right, bottom-left, top-right, top-left)
  • initialOpen: Se o widget deve iniciar aberto (true) ou fechado (false) no modo flutuante
  • hideHeader: Se o cabeçalho do widget deve ser ocultado (true) ou exibido (false)
  • width: Largura do widget no modo flutuante (padrão: 350px)
  • height: Altura do widget no modo flutuante (padrão: 500px)
