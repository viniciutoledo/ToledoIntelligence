ToledoIA Widget - Guia de Incorporação

Integre o assistente ToledoIA em seu site em minutos

Visão Geral

O widget do ToledoIA permite que você ofereça assistência especializada em circuitos eletrônicos aos seus usuários diretamente no seu site. Este guia mostra como incorporar o widget no seu site usando diferentes métodos.

Importante: Você precisa ter uma chave API válida para usar o widget. Você pode criar e gerenciar suas chaves API no painel administrativo.

Método 1: Widget Flutuante

Este método adiciona um pequeno botão flutuante no canto inferior direito do seu site. Quando o usuário clica no botão, o widget do chat se abre.

Passo 1: Adicione o script do ToledoIA ao seu site

<script src="https://toledoia.replit.app/widget.js"></script>

Passo 2: Inicialize o widget com sua chave API

<script> document.addEventListener('DOMContentLoaded', function() { ToledoIAWidget.init({ apiKey: "sua-chave-api-aqui", position: "bottom-right", // Opções: "bottom-right", "bottom-left", "top-right", "top-left" mode: "floating" // Modo flutuante (padrão) }); }); </script>

Método 2: Widget Inline

Este método incorpora o widget diretamente em um elemento do seu site, como uma seção ou div.

Passo 1: Adicione o script do ToledoIA ao seu site

<script src="https://toledoia.replit.app/widget.js"></script>

Passo 2: Crie um elemento container para o widget

<div id="toledoia-container" style="width:100%; height:500px;"></div>

Passo 3: Inicialize o widget com sua chave API no modo inline

<script> document.addEventListener('DOMContentLoaded', function() { ToledoIAWidget.init({ apiKey: "sua-chave-api-aqui", mode: "inline", targetElement: "#toledoia-container", // Seletor CSS do elemento container hideHeader: true // Opcional: oculta o cabeçalho para uma aparência mais integrada }); }); </script>

Parâmetros e Opções

Você pode personalizar o comportamento e a aparência do widget usando os seguintes parâmetros:

Parâmetro Tipo Padrão Descrição
apiKey string - Obrigatório. Sua chave API do ToledoIA.
mode string "floating" Modo de exibição do widget. Opções: "floating" ou "inline".
position string "bottom-right" Posição do botão flutuante. Opções: "bottom-right", "bottom-left", "top-right", "top-left".
targetElement string - Seletor CSS do elemento que conterá o widget (obrigatório para modo "inline").
hideHeader boolean false Se verdadeiro, oculta o cabeçalho do widget, útil para integrações personalizadas.
autoOpen boolean false Se verdadeiro, o widget flutuante abre automaticamente ao carregar a página.

Métodos API

O widget expõe os seguintes métodos que podem ser chamados programaticamente:

// Abrir o widget ToledoIAWidget.open(); // Fechar o widget ToledoIAWidget.close(); // Remover completamente o widget da página ToledoIAWidget.destroy();

Demonstração

Exemplo de Widget Flutuante

Clique no botão abaixo para ver o widget flutuante em ação.

Mostrar Widget Flutuante
<script> function demoFloatingWidget() { // Primeiro, remova qualquer widget existente if (window.ToledoIAWidget) { ToledoIAWidget.destroy(); } // Inicialize o widget no modo flutuante ToledoIAWidget.init({ apiKey: "7b1e783a-5abb-4269-af75-0ca4a0129dd2", position: "bottom-right", mode: "floating", autoOpen: true }); } </script>
