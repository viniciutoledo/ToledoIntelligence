ToledoIA - Exemplo de Incorporação via iframe

Esta página demonstra como incorporar o widget do ToledoIA no seu site usando a tag iframe com o endpoint dedicado /embed/iframe. Este formato é compatível com sistemas LMS como o Curseduca, Moodle e outros.

Importante

Substitua SUA_API_KEY pela chave API do seu widget ToledoIA.

A URL do iframe deve seguir o formato:

https://[seu-dominio]/embed/iframe?key=SUA_API_KEY

Código de Incorporação

Adicione este código HTML onde quiser exibir o widget de chat:

<iframe src="https://seu-dominio.com/embed/iframe?key=SUA_API_KEY" width="100%" height="600" frameborder="0" allow="microphone" style="border: none; width: 100%; height: 600px;" ></iframe>

Embed no Curseduca

Para incorporar no Curseduca, use este formato na opção "Criar Item" > "Link externo":

/embed?url=https%3A%2F%2Fseu-dominio.com%2Fembed%2Fiframe%3Fkey%3DSUA_API_KEY

Atributos importantes do iframe

  • width: Largura do iframe (pode ser em pixels ou porcentagem)
  • height: Altura do iframe (recomendado mínimo de 500px)
  • frameborder: Definir como "0" para remover bordas
  • allow: Inclua "microphone" para permitir entrada de áudio
  • style: Estilos CSS adicionais conforme necessário

Dicas de implementação

O widget se adapta automaticamente ao tamanho do iframe. Para melhor experiência no mobile, use valores responsivos (como 100%) para a largura.

Você também pode customizar a aparência do chat pelo painel administrativo do ToledoIA.
